# Personal Knowledge Base

Персональная база знаний о backend engineering, языках программирования, системном дизайне и инфраструктуре.

Этот репозиторий содержит структурированные заметки, примеры кода, эксперименты и мини-проекты, которые я создаю во время изучения и работы с backend технологиями.

Цель этого репозитория - построить **долгосрочную базу знаний** и задокументировать практический опыт работы с различными инструментами и концепциями.

---

# Philosophy

Этот репозиторий следует нескольким принципам:

• Одна тема = один файл
• Знания написаны своими словами
• Каждая тема должна содержать примеры или объяснения
• Заметки должны быть небольшими и сфокусированными
• Репозиторий растет постепенно

Этот репозиторий **не является копией документации**, а представляет собой структурированную коллекцию учебных заметок и инженерных экспериментов.
