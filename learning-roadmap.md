# Learning Roadmap

Этот план описывает технологии и концепции, необходимые для становления сильным backend-инженером.

---

# Stage 1 — Programming Languages

Фокус на освоении языков программирования, используемых в backend-разработке.

Темы:

* Python advanced features
* Python async programming
* Go concurrency model

Цель:

Понять внутреннее устройство языка и механизмы конкурентности.

---

# Stage 2 — Backend Development

Изучение того, как устроены backend-приложения.

Темы:

* REST API design
* authentication
* authorization
* API versioning
* dependency injection

Цель:

Уметь проектировать масштабируемые API.

---

# Stage 3 — Databases

Глубокое понимание реляционных баз данных.

Темы:

* indexing
* query optimization
* transactions
* isolation levels

Цель:

Уметь писать эффективные и масштабируемые запросы к БД.

---

# Stage 4 — Distributed Systems

Изучение того, как работают большие системы.

Темы:

* caching
* queues
* load balancing
* consistency models

Цель:

Уметь проектировать масштабируемые распределенные системы.

---

# Stage 5 — DevOps

Понимание развертывания и инфраструктуры.

Темы:

* Docker
* container networking
* CI/CD pipelines
* reverse proxy

Цель:

Уметь развертывать и эксплуатировать backend-системы.

---

# Stage 6 — System Design

Изучение того, как проектировать большие системы.

Темы:

* rate limiter
* URL shortener
* job queues
* chat systems

Цель:

Уметь проектировать системы продакшн-уровня.
